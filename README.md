# Landing Page

This project is a responsive landing page created with HTML, CSS, and JavaScript. The goal of the page is to provide a modern and interactive user experience while keeping the design simple and clean.

The website includes smooth scrolling navigation, scroll animations, and a responsive menu that adapts to different screen sizes. It also has a basic contact form validation system to make sure users enter the required information correctly before submitting.

## Features

* Responsive design for desktop and mobile devices
* Smooth scrolling between sections
* Scroll reveal animations
* Responsive navigation menu
* Contact form validation with JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

During this project, I practiced:

* Responsive web design
* Basic form validation
* Organizing JavaScript code more clearly
